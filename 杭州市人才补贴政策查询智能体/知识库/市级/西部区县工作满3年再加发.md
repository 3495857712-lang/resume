# 西部区县工作满 3 年再加发（生活补贴）

> **政策层级**：市级  
> **适用区域**：富阳区、临安区、桐庐县、淳安县、建德市  
> **来源**：[杭州市人社局申请指南](https://hrss.hangzhou.gov.cn/art/2025/3/3/art_1229564773_1849781.html)

## 补贴标准（再次发放，一次性）

| 学历 | 再加发金额 |
|------|------------|
| 本科 | 1 万元 |
| 硕士 | 3 万元 |
| 博士 | 10 万元 |
| 博士（按原标准5万申领过首笔的） | 仍按 **5 万元** 执行 |

## 适用对象

在西部区县领取过「应届毕业生生活补贴」，并在西部区县**连续工作满 3 年**的大学毕业生。

## 核心条件（摘要）

1. 在西部区县领取过应届毕业生生活补贴
2. 仍在西部区县**符合条件的用人单位**连续工作至 **3 年期满**
3. 连续参保，**不含补缴**（含毕业到领取补贴期间在西部区县连续工作时长）
4. 分两笔发放补贴的，通常指**第二笔补贴**领取后的连续工作计算
5. 因择业等原因社保断缴的：给予最长 **6 个月（含）择业期**（从补贴审核通过次月起算），之后在西部区县符合条件的单位又连续工作满 3 年

## 申请渠道

- **亲清在线** → 杭州市应届毕业生生活补贴模块
- 点击「**西部区、县(市)工作满三年再次补贴申请**」

## 注意事项

- 连续工作 3 年的**参保地**需为西部区、县（市）
- 2019-06-03（含）之后毕业且在西部区县领取过首笔补贴的，申请时仍按当前政策执行
- 系统政策有调整历史，以亲清在线当前选项为准
