# 补贴审核、异议处理与常见驳回

> **来源**：杭州市人社局、市住保房管局、亲清在线操作说明  
> **更新**：2026-07-07

## 生活补贴

| 环节 | 规则 |
|------|------|
| 第一笔 | 须**主动申请**（亲清在线 / 杭州人才码生活补贴模块） |
| 第二笔、西部 3 年再加发 | 一般由**系统自动比对发放**；劳务派遣经历等除外 |
| 发放 | 审核通过后拨付至申请人**银行卡** |
| 异议 | 审核未通过后 **3 个月内** 可提出异议处理 |
| 未自动发放 | 可在市民卡 APP 点「第二笔补贴申请」，或由单位在亲清在线代申请 |

### 劳务派遣

- 计入连续参保期间有**劳务派遣经历**的，须由**实际用工单位**通过亲清在线申报
- 劳务派遣人员当前在职的，也需单位申报

## 租房补贴

| 环节 | 规则 |
|------|------|
| 个人申请 | **杭州人才码（青荷码）**，须个人诚信承诺 |
| 单位申请 | **亲清在线**代员工申报 |
| 异议 | 单位经办人在亲清在线上传证明材料 |

### 常见驳回原因与处理

| 驳回提示 | 处理建议 |
|----------|----------|
| 社保单位与申报单位不一致 | 核实后由单位上传**社保缴纳证明**提交异议 |
| 查不到社保 | 参保次月再试；仍不行上传证明异议 |
| 国（境）外学历查不到 | 上传**教育部留服中心**认证证书 |
| 未按规定劳务派遣 | 实际用工单位上传劳务派遣合同、社保证明、派遣协议 |
| 兑付失败 | 核对**本人一类卡**账号，非挂失冻结，限额充足；亲清在线修改信息后异议 |

### 家庭认定

- 申请家庭 = **本人 + 配偶 + 未成年子女**
- **父母有房不影响**申请

---

## 青荷礼包

| 环节 | 规则 |
|------|------|
| 异议 | 系统自动审核未通过，**15 日内**可申请人工复核 |
| 参保当月 | 可能查不到社保，建议**次月**申请 |
