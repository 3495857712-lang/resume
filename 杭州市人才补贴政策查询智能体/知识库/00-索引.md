## 申报平台

| 平台 | 用途 |
|------|------|
| [亲清在线](https://qinqing.hangzhou.gov.cn/) | 生活补贴、租房补贴（单位端） |
| **杭州市民卡 APP → 杭州人才码** | 青年人才认定、生活/租房/青荷礼包 |
| 微信小程序 **「人才杭州」** | 青荷驿站预约 |
| 浙里办 APP | 部分区级补贴（如萧山人才码） |
| 各区人才/人社官网 | 区级配套政策 |

## 统一咨询

- **杭州人社咨询**：0571-96345
- **杭州市人社局**：https://hrss.hangzhou.gov.cn/

## 智能体回答逻辑

1. 判断用户属于哪类人才（应届本科+/大专/高层次人才认定）
2. 判断工作/社保缴纳在哪个区
3. 匹配市级政策 + 区级配套
4. 检查互斥（租房类补贴通常不可重复享受）
5. 给出申请路径与材料要点
